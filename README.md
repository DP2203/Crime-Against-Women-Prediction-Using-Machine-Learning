# Crime-Against-Women-Prediction-Using-Machine-Learning
Crime Against Women Prediction Using Machine Learning
Project Overview

This project is a Machine Learning-based web application developed using Streamlit that predicts crimes against women based on historical crime data.

The application allows users to upload a dataset, select machine learning models, analyze crime patterns, and generate predictions for future crime counts.

The system also includes user authentication and an admin panel to manage registered users.

Features
User Features

User registration and login system

Upload crime dataset (CSV format)

Select features and target variables

Train multiple machine learning models

View model evaluation metrics

Visualize actual vs predicted values

Predict future crime counts (e.g., 2025)

Admin Features

Admin login

View all registered users

Delete users from the system

Monitor application usage

Machine Learning Models Used

The following models are implemented in the system:

Linear Regression

Random Forest Regressor

Decision Tree Regressor

Gradient Boosting Regressor

These models are used to analyze crime data and predict future trends.
